Grid Layout

Use a grid layout to position children at fixed intervals. Unlike a flexible layout, the sizes of children are not considered for layout. Objects are placed in cells in column-row order. The size of the cells is determined by the size of the Flexalon Object Component.

Grid Layout Options

Cell Type Rectangular or Hexagonal cells.
Columns Number of columns.
Rows Number of rows.
Column Direction Axis to create columns and the direction to order child objects.
Row Direction Axis to create rows and the direction to increment the rows when placing child objects.
Column Spacing Adds empty space between columns.
Row Spacing Adds empty space between rows.
Align Aligns each item within its cell.
